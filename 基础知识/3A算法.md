# 3A算法

## 定义

3A算法是 ISP（Image Signal Processor）中最核心的三个自动控制算法：
- **AE (Auto Exposure)**：自动曝光
- **AWB (Auto White Balance)**：自动白平衡
- **AF (Auto Focus)**：自动对焦

这三个算法协同工作，确保图像亮度适中、色彩准确、主体清晰。

## 核心要点

### AE (Auto Exposure) 自动曝光

**目标**：调整曝光参数使图像亮度达到目标值

**控制参数**
| 参数 | 说明 | 调节优先级 |
|------|------|-----------|
| 曝光时间 | 传感器积分时间 | 优先调整 |
| 模拟增益 | Sensor 模拟放大 | 次优先 |
| 数字增益 | ISP 数字放大 | 最后 |
| 光圈 | 镜头通光孔径 | 固定或自动 |

**曝光策略**
```
曝光不足 → 增加曝光时间 → 达到上限 → 增加模拟增益 → 达到上限 → 增加数字增益
曝光过度 → 减少数字增益 → 达到下限 → 减少模拟增益 → 达到下限 → 减少曝光时间
```

### AWB (Auto White Balance) 自动白平衡

**目标**：校正光源色温，使白色物体呈现白色

**核心算法**
| 算法 | 原理 | 适用场景 |
|------|------|----------|
| **灰度世界** | 假设场景平均为灰色 | 一般场景 |
| **完美反射** | 假设最亮点为白色 | 有高光的场景 |
| **白点检测** | 检测落在白区的像素 | 复杂光源 |

**色温范围**
| 色温 (K) | 光源类型 | 颜色倾向 |
|----------|----------|----------|
| 2500-3000 | 白炽灯/烛光 | 暖黄 |
| 4000-5000 | 荧光灯/LED | 中性偏绿 |
| 5500-6500 | 日光 | 中性白 |
| 7000-9000 | 阴天/阴影 | 偏蓝冷 |

### AF (Auto Focus) 自动对焦

**目标**：移动镜头使被摄物体清晰成像

**对焦方式对比**
| 方式 | 原理 | 速度 | 精度 | 弱光表现 |
|------|------|------|------|----------|
| **CDAF** | 对比度检测 | 慢 | 高 | 差 |
| **PDAF** | 相位检测 | 快 | 中 | 一般 |
| **Laser AF** | 激光测距 | 快 | 高 | 好 |

**CDAF 搜索策略**
```
粗搜 (Coarse) → 大步进寻找峰值区域 → 精搜 (Fine) → 小步进精确定位峰值
```

## 3A 协同工作

### 触发条件

| 场景 | 触发算法 | 处理策略 |
|------|----------|----------|
| 场景切换 | AE + AWB | 快速收敛 |
| 点击对焦 | AF + AE | 对焦优先曝光 |
| 低光环境 | AE + AF | 增益优先，激光辅助 |
| 运动跟踪 | AF + AE | 预测对焦，曝光锁定 |

### 收敛流程

```
帧 N:   统计当前帧信息
        ↓
        计算与目标差异
        ↓
帧 N+1: 应用新参数
        ↓
        统计验证
        ↓
        是否收敛？→ 是: 保持 | 否: 继续调整
```

## 传统 vs AI 3A

| 维度 | 传统方案 | AI 方案 |
|------|---------|---------|
| **核心原理** | 统计学假设 | 深度学习 |
| **AE 表现** | 逆光易过曝/欠曝 | 语义理解，优先人脸曝光 |
| **AWB 表现** | 大色块干扰 | 复杂光源准确识别 |
| **AF 表现** | 低反差易拉风箱 | AI 预测追踪 |
| **硬件需求** | 低功耗 | NPU/DSP 高算力 |
| **调试难度** | 手动调参 | 数据训练 |
