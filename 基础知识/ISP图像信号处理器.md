# ISP图像信号处理器

## 定义

ISP（Image Signal Processor，图像信号处理器）是数字成像系统中的核心组件，负责将图像传感器（CMOS/CCD）输出的原始 Bayer RAW 数据转换为高质量、可供显示或存储的 RGB/YUV 图像。ISP 通过一系列精心设计的算法模块，校正传感器和镜头的固有缺陷，优化图像质量。

## 核心要点

### ISP Pipeline 三大阶段

```
┌─────────────────────────────────────────────────────────────────┐
│                      ISP Pipeline 全流程                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐       │
│  │   RAW域处理   │ → │   RGB域处理   │ → │   YUV域处理   │       │
│  └──────────────┘    └──────────────┘    └──────────────┘       │
│                                                                 │
│  • 黑电平校正(BLC)    • 去马赛克        • 色彩空间转换            │
│  • 坏点校正(DPC)      • 自动白平衡(AWB)  • YUV域降噪              │
│  • 镜头阴影校正(LSC)  • 色彩校正矩阵     • 边缘增强(EE)            │
│  • RAW域降噪          • Gamma校正       • 伪色抑制               │
│                       • 色调映射        • 对比度/饱和度调整        │
└─────────────────────────────────────────────────────────────────┘
```

### RAW域处理（传感器校正）

| 模块 | 英文 | 功能 |
|------|------|------|
| **黑电平校正** | BLC (Black Level Correction) | 消除暗电流噪声，确保黑场为零电平 |
| **坏点校正** | DPC (Dead Pixel Correction) | 修复传感器坏点（亮点/暗点） |
| **镜头阴影校正** | LSC (Lens Shading Correction) | 补偿镜头边缘亮度衰减和色偏 |
| **RAW域降噪** | NR (Noise Reduction) | 抑制传感器噪声，防止后续放大 |

### RGB域处理（色彩重建）

| 模块 | 英文 | 功能 |
|------|------|------|
| **去马赛克** | Demosaic / CFA Interpolation | 将 Bayer 单通道数据插值为全彩 RGB |
| **自动白平衡** | AWB (Auto White Balance) | 校正光源色温，使白色物体呈白色 |
| **色彩校正矩阵** | CCM (Color Correction Matrix) | 调整色彩响应，匹配人眼感知 |
| **Gamma校正** | Gamma Correction | 非线性亮度调整，适配人眼和显示设备 |

### YUV域处理（图像增强）

| 模块 | 英文 | 功能 |
|------|------|------|
| **色彩空间转换** | CSC (Color Space Conversion) | RGB → YUV 分离亮度和色度 |
| **YUV域降噪** | NF (Noise Filter) | 分别抑制亮度噪声和色度噪声 |
| **边缘增强** | EE (Edge Enhancement) | 锐化图像边缘，提升清晰度 |
| **伪色抑制** | FCS (False Color Suppression) | 消除去马赛克产生的伪色 |

### 3A算法（闭环控制）

| 算法 | 功能 |
|------|------|
| **AE (Auto Exposure)** | 自动曝光控制，调整曝光时间和增益 |
| **AWB (Auto White Balance)** | 自动白平衡，估计色温并调整色彩 |
| **AF (Auto Focus)** | 自动对焦，分析图像清晰度控制镜头 |

## 关键特性

### 处理域划分
- **RAW域**：Bayer 格式数据，每个像素仅一种颜色
- **RGB域**：全彩数据，每个像素包含 R/G/B 三个通道
- **YUV域**：亮度(Y)与色度(UV)分离，便于压缩和处理

### 数据位深
- 输入 RAW：通常 10-14 bit
- 处理过程：通常 12-16 bit（防止精度损失）
- 输出：8 bit（JPEG）或 10-12 bit（RAW/视频）
