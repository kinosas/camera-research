# ISP（图像信号处理器）

## 定义

ISP（Image Signal Processor，图像信号处理器）是专门用于处理图像传感器输出信号的处理器。它将传感器捕获的原始 RAW 数据转换为可用的图像/视频，是相机系统中承上启下的核心组件。

## 核心要点

### ISP 在相机系统中的位置
```
镜头 → 传感器 → ISP → 编码器 → 存储/显示
              ↓
           自动对焦/曝光算法
```

### 主要处理流程

1. **黑电平校正**（Black Level Correction）
2. **去马赛克**（Demosaic）- 重建完整 RGB
3. **白平衡**（White Balance）- 校正色温
4. **色彩校正**（Color Correction Matrix）
5. **降噪**（Noise Reduction）
6. **锐化**（Sharpening）
7. **伽马校正**（Gamma Correction）
8. **色调映射**（Tone Mapping）- HDR 处理

### RYYB 对 ISP 的特殊要求

**运算量激增**
- 传统 RGGB Demosaic 算法无法直接套用
- 华为 ISP 运算量提升 **5 倍**
- 需要专用 NPU 加速神经网络算法

**算法创新**
- 基于 CNN 的去马赛克（CNN-DM）
- 超过 **3000 组色彩映射模型**
- RAW 域直接还原色彩

**长期优化**
- 华为投入约 **3 年** 调校时间
- 从 P30 到 P40 才基本解决偏色问题
- 持续迭代至 Mate 70 系列

## 关键技术指标

| 指标 | 说明 |
|------|------|
| 处理速度 | 决定连拍性能和视频帧率 |
| 位深支持 | 影响动态范围和色彩精度 |
| 降噪算法 | 影响低光画质 |
| HDR 融合 | 多帧合成能力 |
| AI 加速 | NPU 协处理支持 |

- **索尼** - 部分传感器内置 ISP
